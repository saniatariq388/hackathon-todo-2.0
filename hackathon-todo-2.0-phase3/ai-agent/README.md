# AI Agent for Todo Application

This is the AI agent that uses the Model Context Protocol (MCP) to interact with the todo application backend. It provides a natural language interface for managing tasks.

## Features

- Natural language task creation
- Task listing and management
- Task updates and deletions
- Integration with the MCP server for backend operations

## API

The agent exposes a `/chat` endpoint that accepts a message and returns an AI-generated response. The endpoint expects a Bearer token for authentication.
